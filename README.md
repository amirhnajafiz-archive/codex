# CodeX Golang Example

<br />

## Test

### Build

```shell
go build -o main
```

### Run

```shell
./main -f example.py -e py -i 8
```

<br />

## Flags

|   Flag   | Description    | Example       |
|:--------:|----------------|---------------|
| ```-f``` | Input file     | ```main.py``` |
| ```-e``` | Extension type | ```py```      |
| ```-i``` | File inputs    | ```5```       |
